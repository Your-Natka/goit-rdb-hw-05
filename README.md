# goit-rdb-hw-05

1. Вкладений запит у SELECT

Створити запит, який виведе всі стовпчики таблиці order_details та відповідний customer_id із таблиці orders.
Пошук customer_id потрібно реалізувати через вкладений підзапит у операторі SELECT.

2. Вкладений запит у WHERE

Вивести записи з таблиці order_details, але лише ті, для яких у таблиці orders відповідне поле shipper_id = 3.
Фільтрація повинна бути виконана за допомогою вкладеного підзапиту в операторі WHERE.

3. Вкладений запит у FROM

Створити підзапит у блоці FROM, який вибирає записи з таблиці order_details, де quantity > 10.
На основі цих даних обчислити середнє значення quantity, згрупувавши результат за order_id.

4. Розв’язання задачі №3 через WITH

Переписати попередній запит, використавши тимчасову таблицю temp, створену через WITH.
Якщо MySQL < 8.0 — використовувати аналог, наведений у конспекті.

5. Створення функції

Створити функцію, яка має два параметри і повертає результат ділення першого параметра на другий.
Умови:

Параметри та тип, що повертається — FLOAT;

Перед створенням використати DROP FUNCTION IF EXISTS;

Застосувати функцію до поля quantity таблиці order_details, другий параметр — довільне число.
